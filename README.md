# cs166-project
CS166 Final Project

## How to Run
### Start Database
```
source createPostgreSQL.sh
```

### Create Database
```
source startPostgreSQL.sh
```

### Load Database
This will create tables, indices, and laod data into the database.
```
loadDB.sh
```

### Running Web Interface (GUI)
```
python3 app.py
```

### Stop Database (when done)
```
source stopPostgreDB.sh
```
